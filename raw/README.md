Isi folder `raw` dengan masing-masing data, termasuk video (mp4) dan file anotasi (.pkl) yang terbagi menjadi data _train_, _test_, dan _validation_. Data diunduh dari tautan [https://chalearnlap.cvc.uab.cat/dataset/24/description/](https://chalearnlap.cvc.uab.cat/dataset/24/description/).

Hasil struktur direktori seperti berikut.
```
raw/
├── test/
│   └── ...
├── train/
│   └── ...
├── validation/
│   └── ...
├── annotation_test.pkl
├── annotation_training.pkl
└── annotation_validation.pkl
```
